# Contribution Guide

This repository contains a python command line application for copying Flight Log Files from a DJI Remote Control using the MTP protocol.

## Style
- Format Python code using `black` with default settings.
- Follow PEP8 and use type hints where applicable.

## Checks
- Install dependencies from `requirements.txt`.
- Ensure `python -m py_compile getmyglightlogs.py` runs without errors.
- Run `python getmyglightlogs.py --help` to verify the script launches.

## Documentation
- Update `README.md` if you change or add features.

## Pull Requests
- Include a brief description of what was changed and why.
